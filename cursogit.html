Working directory

Modificado (Modified/Working Directory): El archivo se ha cambiado, pero aún no se ha guardado en el área de preparación ni en la base de datos. Es el estado inicial de edición.

staged area

Preparado (Staged/Area de preparación): El archivo modificado ha sido marcado en su versión actual y está incluido en la próxima instantánea (snapshot) de confirmación, generalmente mediante el comando git add.

committed

Confirmado (Committed/Repositorio Local): Los cambios se han guardado de manera segura en la base de datos local de Git, consolidando la versión con git commit. Volver a la página principal
